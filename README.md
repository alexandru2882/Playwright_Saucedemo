## Playwright Saucedemo

End-to-end tests for Saucedemo using Playwright + TypeScript.

### Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run tests:
   ```bash
   npx playwright test
   ```

### Structure

See `src/pages` for POMs, `tests` for specs, and `.github/workflows/playwright.yml` for CI.
